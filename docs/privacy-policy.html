1. Overview

Quick Test Management for Jira ("the App") is a Forge-native application built entirely on Atlassian's Forge platform. It is designed to help users track test statuses directly within Jira issues. We value your privacy and are committed to protecting your data.

2. Technology Platform

This app is built using Atlassian Forge, Atlassian's next-generation cloud app development platform. Forge apps run entirely within Atlassian's secure cloud infrastructure, ensuring:

  • No External Servers: The app does not connect to any third-party servers or external APIs.
  • Sandboxed Execution: All app code runs in Atlassian's secure, sandboxed environment.
  • Native Integration: The app integrates directly with Jira using Atlassian's official APIs.

3. Data Collection and Storage

The App stores the following data:

  • Test Status: The current status (Pass, Fail, Blocked, Untested, In Progress) of a test case linked to a Jira issue.
  • Test Notes: Optional notes entered by the user regarding the test case.
  • History Logs: Timestamps and status changes for audit history.

4. Data Access

The data stored by the App is accessible only to:

  • Users with access to the specific Jira issue where the data was created.
